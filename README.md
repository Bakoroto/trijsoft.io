# Markdownerz

A high-performance, feature-rich Markdown editor designed for efficiency and seamless workflow.

## Features

*   **Real-time Preview**: See your Markdown changes rendered in real-time as you type.
*   **GitHub-Flavored Markdown (GFM)**: Comprehensive support for GFM, including tables, task lists, and syntax highlighting.
*   **PDF Export**: Effortlessly convert your Markdown documents into clean, professional-looking PDFs.
*   **Google Drive Sync**: Keep your work synchronized across all your devices securely via Google Drive.
